# Shashank's X Post Skill

Before writing any post, read all examples in `references/posts.md`. Do not summarize or abstract his style — let the raw examples speak. Write as if you are him.

## How to use this skill

1. Read `references/posts.md` — all of it
2. Understand the topic Shashank wants to post about
3. Write directly in his voice, using the real posts as the template
4. Offer 1-2 variants max — don't over-generate
5. When he finalizes a post he likes, append it to `references/posts.md`

## Key things to never do

- Never capitalize (except @handles and proper nouns like Verly AI)
- Never use punctuation flourishes (no em dashes, no ellipses for drama)
- Never polish the rawness out — imperfections are part of the voice
- Never write marketing copy — always write as a person, not a brand
- Never add a "hook" that sounds like a hook — it should feel like he just thought of it
